## Aula Package API Reference

When trying to understand the `aula` PyPI package, consult **`docs/aula-package-api.md`** — it documents every public method, data model, enum, exception, and constant in `aula==0.6.1`.

## Project Conventions

- **Integration domain:** `hass_aula`
- **Manifest:** `custom_components/hass_aula/manifest.json`
- **Scripts:** `scripts/` (develop, lint, setup, release)
- **Release:** `scripts/release <version> [--dry-run]`
